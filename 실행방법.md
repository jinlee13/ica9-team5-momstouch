# 똑똑한 엄마 — 로컬 실행 방법

## 한 번만 하면 되는 것 (이미 완료)
```
npm install
```

## 매번 실행할 때
```bash
# 이 폴더에서 터미널 열고:
npm run dev
```
→ 브라우저에서 http://localhost:3000 열기

## 화면 구성
1. **온보딩** (/) — 아이 생년월일 입력
2. **홈 대시보드** (/home) — 지금/곧/아직 탭 + 카테고리 필터
3. **상품 상세** (/products/[id]) — DDOK 발달 이론 + 쿠팡 링크
4. **체크리스트** (/checklist) — 구매완료/보류/생략 관리

## 데이터 현황
| 카테고리 | 상태 | 담당 |
|---------|------|------|
| 발달 이론 (DDOK) | ✅ 통합 완료 | 이승일, 심은민 |
| 안전용품 | ✅ 카테고리 반영 | 하성준 |
| 발달교구·놀이 | ✅ 카테고리 반영 | 김주연 |
| 수면·위생 | ✅ 카테고리 반영 | 이재진 |
| 인플루언서 데이터 | ✅ 72개 lifecycle_rules 기반 | 황성철 |
| 먹기(음식) | ⏳ 데이터 대기 중 | 안혜린 |

## 추후 할 일
- 쿠팡 파트너스 URL 입력: `data/products.json`에서 `"coupangUrl": null` 찾아서 실제 URL 입력
- 안혜린 데이터 받으면 먹기 카테고리 상품 추가
