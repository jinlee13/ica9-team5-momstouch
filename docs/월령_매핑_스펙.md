# 카테고리-월령 매핑 테이블 생성 프롬프트

## 작업 개요

`ddokddok-mom.vercel.app/browse` 사이트의 육아용품 데이터베이스에 **월령별 추천 필터** 기능을 추가한다.
각 상품에 `recommended_age_min`(개월, 이상)과 `recommended_age_max`(개월, 미만, `null`이면 상한 없음) 컬럼을 부여하는 매핑 로직을 구현한다.

---

## 매핑 규칙 (대분류 → 중분류 → 조건 순서)

> **우선순위**: 아래 규칙은 위에서 아래 순서로 적용하고, 먼저 매칭된 규칙을 따른다.
> 모든 문자열 비교는 제품명(title) 기준이며 대소문자 무관하게 처리한다.

#### ⚠️ 전체 공통 재분류 규칙 1: 구강케어 강제 이동

**대분류·중분류에 관계없이**, 제품명에 아래 키워드 중 하나라도 포함되면 **무조건 중분류를 '구강케어'로 변경**한다:

- `치약`, `칫솔`, `치실`, `구강`
- `불소` (단, `무불소`만 있고 위 4개 키워드가 없으면 제외)

> 예: 베이비케어에 등록된 '아기 치약', 위생용품에 등록된 '구강세정기' 모두 구강케어로 이동한다.

## 구현 지침

### 컬럼 스펙

```python
recommended_age_min: int   # 0 이상, 단위: 개월 (이상, 포함)
recommended_age_max: int | None  # 단위: 개월 (미만, 미포함), None이면 상한 없음
```

### 매핑 함수 구조 (의사코드)

```python
def get_age_mapping(category_main, category_sub, product_name):
    name = product_name.lower()

    # 1. 중분류 재분류
    category_sub = reclassify_subcategory(category_sub, name)

    # 2. 매핑 규칙 적용
    if category_main == "자기·위생":
        if category_sub == "기저귀":
            return map_diaper(name)
        elif category_sub == "구강케어":
            return map_oral_care(name)
        elif category_sub == "베이비케어":
            return map_baby_care(name)
        # ... 나머지 중분류

    elif category_main == "외출·안전":
        if category_sub == "유모차":
            return map_stroller(name)
        # ...

    # 기본값
    return (0, None)
```

### 재분류 함수 로직

```python
def reclassify_subcategory(category_sub, name):
    oral_keywords = ["치약", "칫솔", "치실", "구강"]

    # ① 구강케어 강제 이동 (중분류 무관 — 최우선 규칙)
    # 치약·칫솔·치실·구강 중 하나라도 포함되면 무조건 구강케어
    if any(k in name for k in oral_keywords):
        return "구강케어"
    # '불소'가 있고 '무불소'가 아닌 경우도 구강케어
    if "불소" in name and "무불소" not in name:
        return "구강케어"

    # ② 구강케어 → 베이비케어 역이동 (구강 키워드가 전혀 없는 경우)
    # (①에서 걸리지 않았다는 뜻이므로, 기존 구강케어에 잘못 분류된 항목 정리)
    if category_sub == "구강케어":
        return "베이비케어"

    # ③ 유모차 → 아기띠·캐리어 또는 베이비케어 이동
    if category_sub == "유모차":
        if any(k in name for k in ["캐리어", "백팩", "힙시트"]):
            return "아기띠·캐리어"
        if "유모차" not in name:
            return "베이비케어"  # 유모차 키워드 없으면 베이비케어로

    return category_sub
```
